# Sources de réputation utilisées par SupSécu

SupSécu télécharge des listes publiques complètes puis effectue toutes les recherches de domaines localement. Une présence dans une liste est présentée comme un signalement de sécurité, pas comme une preuve absolue de fraude.

## Phishing.Database

- Projet : <https://github.com/Phishing-Database/Phishing.Database>
- Fichier utilisé : `phishing-domains-ACTIVE.txt`
- Licence indiquée par le projet : MIT

## CERT Polska — liste d’avertissements

- Présentation officielle : <https://cert.pl/lista-ostrzezen/>
- Fichier utilisé : <https://hole.cert.pl/domains/v2/domains.txt>

## PhishDestroy

- Présentation du jeu de données : <https://phishdestroy.io/dataset>
- Projet de la liste : <https://github.com/phishdestroy/destroylist>
- Fichier utilisé : `domains.txt`
- Le projet publie son code sous licence MIT et décrit les données comme librement réutilisables selon les conditions indiquées dans son dépôt et sa page de jeu de données.

Les noms de projets restent la propriété de leurs titulaires respectifs. SupSécu n’est affilié à aucune de ces organisations.
